# CSharp-DataTypes

## 📖 Overview

This repository contains beginner-friendly examples of C# data types using a Console Application. Each data type is implemented in a separate class with clear comments to help understand its purpose and usage.

This project is part of my C# learning journey as I build a strong foundation for .NET development.

---

## 📚 Concepts Covered

- int
- float
- double
- decimal
- long
- char
- bool
- string
- var

## ▶️ How to Run

1. Clone this repository.
2. Open the project in Visual Studio 2022.
3. Build the solution.
4. Press **Ctrl + F5** to run the application.
